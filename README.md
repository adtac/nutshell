# nutshell

![image](https://i.imgur.com/c6CdeAi.png)

**nutshell** is a lightweight command shell with an extremely low
memory footprint.

## Compiling

```bash
$ gcc main.c -o nutshell
```

## How to run it

```bash
$ ./nutshell
```
